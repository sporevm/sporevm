# SporeVM v1.4.0

Date: unreleased

## Highlights

- Simplifies CLI command forms across `run`, `create`, and `exec`: shell
  commands are accepted by default, while `-- <argv...>` keeps exact argv.
- Adds commandless `spore run --from SPORE` attach, the shorter
  `spore attach SPORE` shorthand, `spore ps` as an alias for `spore ls`, terse
  lifecycle status output, and detached startup commands for `spore create`.
